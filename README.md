📚 Book Library Manager
---
Overview 
A modern React + Tailwind CSS web application that allows users to explore books, track reading progress, and manage their personal library.
---
🚀 Features
🔍 Search books using Open Library API
📖 View book details (title, author, cover)
📊 Track reading progress
🧠 Smart UI with responsive design
📱 Fully responsive (mobile, tablet, desktop)
⚡ Fast and dynamic user experience
---
🛠️ Tech Stack
Frontend: React (Vite)
Styling: Tailwind CSS
State Management: React Hooks (useState, useEffect)
API: Open library API
